Holoseat Firmware README

Library Dependencies
--------------------
The Holoseat firmware depends on two external libraries: Bounce2 (v2.2.0) and ArduinoJson (v5.11.2).  You can install both of them from the Arduino IDE's Library Manager.

1. Open the Holoseat firmware sketch (holoseat-firmware.ino) in the Arduino IDE
2. Select Sketch > Include Library > Manage Libraries from the Arduino IDE menu
3. For each dependencie, search for and install the library
